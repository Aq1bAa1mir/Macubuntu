# 🍏 Macbuntu

Macbuntu is a custom Ubuntu LTS-based ISO engineered to deliver a macOS-like desktop experience using a heavily customised GNOME environment.

It is designed for users who want the aesthetics and workflow of macOS while maintaining the flexibility, performance, and open ecosystem of Linux.

---

## 📦 Overview

Macbuntu provides a pre-configured system built on a stable Ubuntu LTS base. The ISO integrates theming, layout, and usability enhancements directly into the operating system, eliminating the need for manual setup after installation.

The goal is to offer a consistent, polished desktop experience with minimal overhead.

---

## 🧱 System Base

- **Base Distribution:** Ubuntu LTS  
- **Desktop Environment:** GNOME (customised)  
- **Display Manager:** GDM (themed)  
- **Package Manager:** APT  

The system remains fully compatible with official Ubuntu repositories and standard package management workflows.

---

## 🎨 Desktop Customisation

Macbuntu applies deep customisation across the GNOME stack to replicate macOS design patterns:

- Custom GTK and GNOME Shell themes  
- macOS-style icon pack and cursor set  
- Dock-based application launcher (replacing traditional taskbar)  
- Top panel configured to mimic macOS menu bar behaviour  
- Window controls styled and positioned to match macOS conventions  
- Refined animations and transitions for smoother UX  

All configurations are pre-applied at the ISO level.

---

## 🚀 Features

- Fully pre-themed macOS-style interface  
- Consistent UI/UX across system applications  
- Lightweight configuration with optimised performance  
- Ready-to-use environment after installation  
- No post-install scripts or manual tweaks required

## 💻 Installation

### 1. Download ISO
Download the Macbuntu ISO from the repository or release source. https://archive.org/details/Macbuntu

### 2. Create Bootable Media
Use a tool such as:
- Rufus (Windows)
- Balena Etcher (Cross-platform)

### 3. Boot into Installer
- Insert USB drive  
- Boot from USB via BIOS/UEFI  

### 4. Install System
Follow the standard Ubuntu installation process.
